# homebrew-tap

Homebrew formulas for installing apps & tools.

## How To Use

Add repository:

```sh
brew tap brew-tools/tap
```

Install a formula:

```sh
brew install <formula>
```

Upgrade:

```sh
brew upgrade <formula>
```
